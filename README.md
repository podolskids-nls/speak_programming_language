# Speak — язык программирования на английском

**Speak** — это динамически типизированный интерпретируемый язык с синтаксисом, похожим на обычный английский. Вместо символов `+`, `-`, `{` используются слова: `plus`, `minus`, `if`, `set`. Язык задуман для людей без опыта программирования.

## Для кого

- Начинающие, которым проще читать «set x to 10», чем `x = 10`
- Обучение основам: переменные, условия, циклы, функции
- Быстрые скрипты и эксперименты

## Как запустить

Требуется [Go 1.21+](https://go.dev/dl/).

```bash
go run main.go examples/hello.speak
```

## Как запустить тесты

```bash
go test ./...
```

## Краткий справочник синтаксиса

| Действие | Синтаксис |
|----------|-----------|
| Комментарий | `-- текст` |
| Переменная | `set x to 10` |
| Вывод | `print x` или `print "текст"` |
| Сложение | `a plus b` |
| Вычитание | `a minus b` |
| Умножение | `a times b` |
| Деление | `a divided by b` |
| Условие | `if x is greater than 5:` … |
| Иначе | `else:` |
| Цикл N раз | `repeat 3 times:` … |
| Цикл while | `while i is less than 5:` … |
| Функция | `define greet with name:` … |
| Вызов | `call greet with "Bob"` |
| Возврат | `return x times x` |

Блоки кода выделяются **отступом** (4 пробела или таб), после заголовка ставится `:`.

## Примеры

В папке `examples/` — **10 программ** (8 рабочих и 2 для проверки ошибок). Запуск любой:

```bash
go run main.go examples/<имя>.speak
```

| Файл | Что показывает |
|------|----------------|
| `hello.speak` | минимальный вывод |
| `variables.speak` | переменные, числа, конкатенация строк |
| `conditions.speak` | `if` / `else`, несколько условий подряд |
| `loops.speak` | `repeat … times` и `while` |
| `functions.speak` | `define`, `call`, `return` |
| `bonus.speak` | условие и арифметика в «бизнес»-сценарии |
| `letter.speak` | функция со строками, без сложной математики |
| `factorial.speak` | рекурсия |
| `error_test_1.speak` | ошибка: неизвестная переменная |
| `error_test_2.speak` | ошибка: деление на ноль |

---

### hello.speak

Первая программа: один `print`.

**Вывод:**
```
Hello, World!
```

---

### variables.speak

Присваивание чисел и строк, вывод значений, склейка строк через `plus`.

**Вывод:**
```
42
Speak
3.14159
Hello Speak
```

---

### conditions.speak

Проверка возраста (`if` / `else`), затем цепочка независимых условий для оценки `score`.

**Вывод:**
```
Adult
Good
```

(«Excellent» и «Fail» не печатаются: 85 не больше 90 и не меньше 50.)

---

### loops.speak

`repeat 5 times` печатает строку пять раз; `while` выводит числа от 1 до 5.

**Вывод:**
```
hello
hello
hello
hello
hello
1
2
3
4
5
```

---

### functions.speak

Функция `greet` с побочным эффектом (`print`), функция `square` с `return`; результат вызова сохраняется в переменную.

**Вывод:**
```
Hello Alice
Hello Bob
49
```

---

### bonus.speak

Если стаж больше 2 лет — бонус 10% от зарплаты, иначе сообщение об отсутствии бонуса.

**Вывод:**
```
Bonus is:
5000
```

---

### letter.speak

Функция `build_greeting` возвращает строку; печать заголовка, текста и подписи письма.

**Вывод:**
```
Dear Alice
Thank you for your help.
Best regards
```

---

### factorial.speak

Рекурсивный расчёт факториала: база `n is equal to 0`, иначе `n times call factorial with n minus 1`.

**Вывод:**
```
1
1
120
3628800
```

(факториалы 0, 1, 5 и 10.)

## Проверка ошибок

Два примера намеренно содержат ошибки. Интерпретатор печатает сообщение с **номером строки** и завершает работу с кодом **1** (`exit status 1`).

### error_test_1.speak

Обращение к переменной, которая не была объявлена:

```speak
print total
```

**Вывод (stderr):**
```
Error at line 1: Unknown variable 'total'
```

---

### error_test_2.speak

Деление на ноль при вычислении выражения:

```speak
set x to 10 divided by 0
print x
```

**Вывод (stderr):** (строка `print x` не выполняется)

```
Error at line 1: Cannot divide by zero
```

Другие типы сообщений (синтаксис, несовместимость типов) — в разделе «Обработка ошибок» в [SPEC.md](SPEC.md).

## Структура проекта

```
├── main.go           — точка входа
├── lexer/            — лексический анализ
├── parser/           — синтаксический анализ (AST)
├── interpreter/      — выполнение программы
├── errors/           — сообщения об ошибках
├── runner/           — пайплайн lexer → parser → interpreter
├── examples/         — примеры программ (.speak)
├── tests/            — автотесты
├── README.md
└── SPEC.md           — полная спецификация языка
```

Подробности грамматики и типов — в [SPEC.md](SPEC.md).
